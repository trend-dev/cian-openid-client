# cian-openid-client

#### Установка/запуск

1. Установить пакеты `npm i`. _Версия node v16.15.0_
2. Создать файл config.ts из example.config.ts и заполнить
3. Запуск с nodemon

```
cd src
npx nodemon ./main.ts
```

4. Открыть [http://localhost:4200](http://localhost:4200/cian?cian_house_id=zhk-dlya-nastya)
